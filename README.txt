Tic Tac Toe
A Tic Tac Toe program made for 8th Light, should be unbeatable and win whenever it can.




SRC
	Main -- Only contains the main method, calls Board

	Board -- Sets up the GUI and contains all user interface, calls Computer

	Computer -- Contains all the computers logic, works with the Board setup by Board

JAR
	Contains TicTacToe.jar, an executable of the game.
